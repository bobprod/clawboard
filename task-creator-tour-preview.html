🛠️ Créer une nouvelle mission
creator-header bottom

Ce formulaire te permet de configurer et de lancer une mission IA complète. Chaque champ influence directement le comportement de l'agent.

Suis ce guide pour créer ta première tâche optimisée en moins de 2 minutes.

1
✏️ Nommer ta tâche
creator-name bottom

Le nom est l'identifiant humain de ta tâche. Sois précis et orienté action.

💡 Bonnes pratiques
✅ Analyser logs système — serveur prod
✅ Morning briefing — résumé actualités IA
❌ Tâche 1 ou Test

Seul champ obligatoire pour valider le formulaire.

2
📝 Écrire un bon prompt
creator-instructions bottom

C'est le cerveau de ta mission. L'agent se basera entièrement sur ces instructions pour agir.

📐 Structure recommandée
1. Contexte — qui es-tu, quel est ton rôle
2. Tâche principale — ce que l'agent doit faire
3. Format de sortie — bullet list, JSON, résumé…
4. Contraintes — ton, longueur, langue
3
✨ Amélioration automatique IA
creator-enhance left-end

Ce bouton réécrit et enrichit ton prompt automatiquement grâce à l'IA.

⚡ Comment l'utiliser
1. Écris un prompt simple, même incomplet
2. Clique sur ce bouton ✏️
3. Le prompt est restructuré et optimisé
4. Ajuste si nécessaire avant de valider

Idéal pour partir d'une idée brute et obtenir un prompt professionnel.

4
🧩 Skill & Modèle LLM
creator-skill-model top
🔧 Skill

Un skill est un plugin spécialisé qui donne à l'agent des capacités étendues. Ex : inbox-monitor, morning-briefing. Laisse vide pour un agent généraliste.

🤖 Choix du modèle
⚡ Nemotron Ultra 253B — tâches complexes, raisonnement
🟣 Claude Sonnet 4.6 — polyvalent, fiable, rapide
🔵 Gemini 2.5 Flash — vitesse, faible coût
🟢 Qwen 2.5 local — aucune API externe
5
⚙️ Agent & Timeout
creator-agent-timeout top
🤝 Agent

Identifiant de l'instance d'agent qui va exécuter cette tâche. Laisse main par défaut ou spécifie un agent dédié pour une architecture multi-agents.

⏱️ Timeout
• 5–15 min — réponses rapides, questions simples
• 30 min — analyse standard (défaut recommandé)
• 60–120 min — tâches longues, multi-étapes
Au-delà du timeout, la tâche est marquée comme échouée.
6
📡 Canal de livraison
creator-canal top

Où veux-tu recevoir le résultat de ta tâche ? L'agent enverra sa réponse finale sur ce canal.

📱 Telegram — Chat ID ou @username
💬 Discord — ID du salon texte
📲 WhatsApp — numéro au format +33...
🔗 Webhook — URL HTTPS de ton serveur

Laisse vide pour consulter uniquement dans l'interface.

7
🎯 Définir des objectifs
creator-objectives top

Les objectifs sont des critères de succès mesurables que l'agent garde en tête pendant son exécution.

📌 Exemples d'objectifs
• Identifier les 3 erreurs critiques dans les logs
• Résumer en moins de 150 mots
• Fournir au moins 5 recommandations actionnables

Facultatif mais fortement recommandé pour les tâches complexes.

8
🔒 Pré-instructions globales
creator-preinstructions top

Par défaut, le system prompt global est injecté en tête de chaque tâche.

Activé (défaut) — l'agent hérite du comportement global
Désactivé — démarre sans contexte global, uniquement tes instructions

Désactive uniquement pour les agents nécessitant un comportement radicalement différent.

9
🚀 Créer & Lancer
creator-actions top

Deux options pour finaliser ta mission :

💾 Créer
Sauvegarde sans lancer. Idéal pour préparer des templates ou planifier.
▶️ Créer & Lancer
Sauvegarde et démarre immédiatement. Redirige vers le détail en temps réel.
